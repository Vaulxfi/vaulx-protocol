The brief · one page · for the team

A Solana credit layer for the world's asset-rich, credit-excluded.

Vaulx is the protocol where the smart contract refuses to release a stablecoin loan until a licensed vault has the asset. That refusal — enforced in Anchor account constraints the Solana runtime cannot bypass — is the moat. Everything around it (KYC, wallet, off-ramp, liquidity, oracles) is composed from Solana's strongest protocols, not rebuilt.

01

The moat — a two-key custody gate.

Every other RWA protocol claims custody enforcement. We encode it. The Anchor program's disburse instruction has one invariant. Two release conditions. Both must be true. The Solana runtime does the enforcing, not us.

That is the 18–24 month replication gap. Competitors can fork the UI in a weekend. They cannot fork the signed relationships with Brinks, Prosegur, Loomis, and a BACEN-licensed SCD partner — and without those, the gate has nothing to gate.

// Anchor — Vault Program
pub fn disburse(ctx: Context<Disburse>) -> Result<()> {
  require!(

  // transfer USDC →
}
